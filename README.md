# Mozilla BBSR Website
This is the official repo for Mozilla BBSR Club's website. Fully made using React.js and firebase. All events will be updated 
here.

## Routes
- / - Homepage
- /about - About Us page
- /events - Events page
- /contact - Get in touch page
- /login - Member's login
